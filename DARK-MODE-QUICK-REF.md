# 🌙☀️ Dark Mode vs Light Mode - Quick Reference

## 📍 Theme Toggler Location
**Top Navbar (Right side)**
```
Navbar: [Logo] [Menu] [Language] [☀️/🌙] ← Theme button here
```

---

## 🎨 Color Comparison

| Element | Light Mode | Dark Mode |
|---------|-----------|-----------|
| **Main Background** | #F5EFE6 (Cream) | #0f0a07 (Deep Black) |
| **Card Background** | #FFFFFF (White) | #1a1510 (Dark Brown) |
| **Text (Primary)** | #3E2C23 (Dark Brown) | #f5f5f5 (Light) |
| **Text (Secondary)** | #666666 (Gray) | #cccccc (Light Gray) |
| **Navbar** | 95% Transparent Cream | 95% Transparent Dark |
| **Accent** | #C8A27A (Brown) | #C8A27A (Brown) - Same |
| **Gold** | #D4AF37 (Gold) | #D4AF37 (Gold) - Same |

## 🖱️ Quick Usage

### To Toggle Theme:
1. **Click sun/moon icon** in top-right navbar
2. **Page automatically changes** to dark/light mode
3. **Your choice is saved** (next visit = same theme)

### Icons:
- ☀️ **Sun** = Light mode active
- 🌙 **Moon** = Dark mode active

✅ **Smooth Transitions** - 0.3s fade between themes

## 💾 Storage

**Key:** `emoraTheme`  
**Values:** `"light"` or `"dark"`  
**Browser:** Local storage (survives refresh)

---

## 🌐 Which Mode to Use?

### Use **Dark Mode** if:
- 🌙 It's nighttime
- 👀 You prefer dark backgrounds
- 🔋 Want to save battery (OLED)
- 💎 Like the elegant dark aesthetic

### Use **Light Mode** if:
- ☀️ It's daytime
- 📖 Prefer light backgrounds
- ✨ Want classic luxury look
- 👁️ Need better readability
